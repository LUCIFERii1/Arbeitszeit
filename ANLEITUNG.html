📱 Arbeitszeiterfassung als App installieren

Zwei Wege – lokal oder im Web – beide kostenlos und ohne Konto

📁 Dateien im ZIP
arbeitszeit-pwa/ index.html ← die App selbst manifest.json ← App-Name, Icon, Farben sw.js ← Offline-Unterstützung icons/ icon-192.png ← App-Icon icon-512.png ← App-Icon (groß)
⚠ Alle Dateien müssen im selben Ordner bleiben. Nicht einzeln verschieben.
🖥 Weg 1: Lokal auf PC / Mac Einfachster Weg

Funktioniert ohne Internet-Verbindung. Du brauchst nur einen einfachen lokalen Webserver – dein Browser öffnet file://-Dateien ohne Service Worker.

Option A – Python (meist vorinstalliert):

  1. ZIP entpacken → in den Ordner arbeitszeit-pwa navigieren
  2. Terminal / Eingabeaufforderung im Ordner öffnen
  3. Diesen Befehl eingeben:
python3 -m http.server 8080
  1. Browser öffnen → http://localhost:8080
  2. Chrome/Edge: in der Adressleiste erscheint ein Installieren-Symbol ⊕ → klicken → fertig!
✓ Die App erscheint dann im Startmenü / Dock wie eine normale App.
── oder ──

Option B – Node.js:

npx serve .
── oder ──

Option C – VS Code Extension:

  1. Extension "Live Server" installieren
  2. index.html öffnen → unten rechts auf Go Live klicken
  3. Browser öffnet automatisch → installieren wie oben
☁ Weg 2: GitHub Pages (kostenlos, weltweit erreichbar) Empfohlen

Kostenloser Webspace von GitHub. Einmal hochladen – dann überall per Link + App installierbar. Du brauchst ein kostenloses GitHub-Konto.

  1. Kostenloses Konto erstellen: github.com
  2. Neues Repository erstellen (grüner Button) → Name z.B. arbeitszeit
  3. Alle 5 Dateien hochladen (Upload files → Dateien reinziehen)
  4. Im Repository: Settings → Pages → Branch: main → Save
  5. Nach ~1 Minute erreichbar unter:
    https://DEIN-NAME.github.io/arbeitszeit/
  6. Diese URL im Smartphone-Browser öffnen → „Zum Startbildschirm hinzufügen"
✓ Auf iPhone: Safari → Teilen-Symbol → „Zum Home-Bildschirm"
✓ Auf Android: Chrome → Menü (⋮) → „App installieren"
⚠ Deine Daten bleiben lokal im Browser gespeichert (LocalStorage). Sie werden nie auf GitHub hochgeladen.
⚡ Weg 2b: Netlify Drop (noch einfacher)
  1. Gehe zu app.netlify.com/drop
  2. Den kompletten Ordner arbeitszeit-pwa per Drag & Drop reinziehen
  3. Fertig – du bekommst sofort eine URL wie https://zufälliger-name.netlify.app
  4. Diese URL auf dem Smartphone öffnen und als App installieren
✓ Kein Konto nötig für den ersten Upload. Kostenlos.
